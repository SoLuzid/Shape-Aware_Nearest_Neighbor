# Shape-Aware Nearest Neighbor

A little project I made in Python to have a better nearest neighbor.\
This is mostly a proof of concept and it's not ultra amazing, but have fun with it.

You need NumPy and Pillow btw.\
Uhh, you use it like this:

```bash
python SANN.py yourfileinput.png youroutputfile.png 420 69
```
